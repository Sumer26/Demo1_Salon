Luxe & Layered Salon Demo Website

This is a static 5-page HTML/CSS salon website demo.

Pages:
- index.html
- services.html
- about.html
- gallery.html
- contact.html

How to use:
1. Open index.html in your browser.
2. Keep all files in the same folder so the links and CSS work.
3. Replace demo name, photos, pricing, address, and contact details for real clients.
4. The contact form is static for demo purposes. It can later be connected to Formspree, Netlify Forms, email service, or a booking platform.

Created as a starter client demo template.
